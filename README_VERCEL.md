# 🚀 Contabilease - Deploy no Vercel

## ⚡ Deploy Rápido

### 1. Verificação Pré-Deploy
```bash
npm run vercel-check
```

### 2. Deploy Automático
- Conecte seu repositório no [Vercel](https://vercel.com)
- Configure as variáveis de ambiente
- Deploy automático a cada push
